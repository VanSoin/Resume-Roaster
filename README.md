# 🔥 Resume Roaster

An AI-powered resume analyzer that gives brutally honest feedback on your resume.

## What it does
- Scores your resume on Clarity, Impact, and ATS Compatibility
- Gives 3 specific actionable improvements
- Powered by Llama 3.3 70B via Groq API

## Tech Stack
- Python
- Streamlit
- Groq API (Llama 3.3 70B)

## How to run locally

1. Clone the repo
   
   git clone https://github.com/YOURNAME/resume-roaster.git

2. Install dependencies
   
   pip install -r requirements.txt

3. Create a .env file and add your Groq API key
   
   GROQ_API_KEY=your_key_here

4. Run the app
   
   streamlit run app.py

## Note
Never commit your .env file. Your API key should stay local only.
